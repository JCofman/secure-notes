# secure-notes using next.js

> [Secure-Notes](https://secure-notes.vercel.app/) example of an notes taking app

## Install and run

```sh
npm install
npm run dev

# or

yarn install
yarn dev

```

## Run tests

```sh
npm run test

# or

yarn test

```

## Run e2e test

```sh
npm run e2e

# or

yarn e2e

```
